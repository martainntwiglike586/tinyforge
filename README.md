## 📋 What is tinyforge?

tinyforge is a small program that teaches itself to write code. It works right on your laptop without needing an internet connection or help from other models. There is no need for cloud services, human feedback, or large teacher models. tinyforge improves its coding skills by itself as you use it.

You don’t have to be a programmer to use tinyforge. It runs on your Windows PC and handles all the complex work for you.
